# Case Técnico de Data Analysis – iFood

Este repositório contém a solução do case técnico de Data Analysis do iFood, incluindo:
- Notebook de processamento e análise de dados (`notebooks/case_ifood.ipynb`)
- Relatório final em PDF (`reports/relatorio_final.pdf`)

## 1. Estrutura do projeto

```text
ifood-case/
├── README.md
├── requirements.txt
├── .gitignore
├── notebooks/
│   └── case_ifood.ipynb
├── src/
│   ├── etl.py
│   ├── ab_test.py
│   ├── segmentation.py
│   └── finance_roi.py
├── data/
│   ├── raw/
│   └── processed/
└── reports/
    └── relatorio_final.pdf
```
# 2. Como executar
pip install -r requirements.txt
jupyter notebook notebooks/case_ifood.ipynb
